# CardLens AI

앞면과 뒷면을 자동 촬영한 뒤 AI가 카드/팩 식별, 예상 등급, 가격 범위, 진품 가능성, 재포장 위험을 한 번에 반환하는 iPhone용 웹앱입니다.

## 중요
GitHub Pages만으로는 비밀 API 키를 안전하게 보관할 수 없습니다. 이 버전은 **Vercel**에 배포해야 합니다.

## 배포
1. 이 폴더의 파일 전체를 새 GitHub 저장소에 업로드합니다.
2. Vercel에서 `Add New Project` → 해당 저장소를 Import합니다.
3. Vercel 프로젝트의 `Settings → Environment Variables`에 아래 값을 추가합니다.
   - 이름: `OPENAI_API_KEY`
   - 값: 본인의 OpenAI API 키
4. 필요하면 `OPENAI_MODEL`에 사용할 비전 모델명을 설정합니다. 미설정 시 `gpt-5.6-luna`를 사용합니다.
5. Deploy를 누른 후 발급된 `https://...vercel.app` 주소를 iPhone Safari에서 엽니다.
6. 카메라 권한을 허용합니다.

## 사용법
- 기본은 카드 가이드입니다. 카드팩은 `가이드: 카드` 버튼을 눌러 `가이드: 카드팩`으로 바꿉니다.
- 물건을 틀 안에 넣고 멈추면 앞면이 자동 촬영됩니다.
- “뒤집어 주세요” 안내 후 뒷면을 넣고 멈추면 자동 촬영 및 AI 분석이 시작됩니다.
- 자동 촬영이 어려우면 `지금 촬영`을 누릅니다.

## 한계
- 사진만으로 진품 및 PSA 등급을 확정할 수 없습니다.
- 가격은 AI 추정 범위이며 실시간 판매 완료가를 보장하지 않습니다.
- 실제 서비스에서는 사용자 인증, 호출 제한, 이미지 저장 정책, 가격 데이터 공급자를 추가하는 것이 좋습니다.
